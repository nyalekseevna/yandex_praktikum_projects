
## Описание проектов

Данные проекты были выполнены в ходе обучения в Яндекс.Практикуме, профессии "Аналитик данных".

| Название проекта | Описание | Используемые библиотеки | 
| :---------------------- | :---------------------- | :---------------------- |
| [Анализ вакансий ручных тестировщиков для QA–факультета](https://github.com/nyalekseevna/yandex_praktikum_projects/tree/master/analytics%20of%20testers%20vacancies)| | Выявила топ самые частых требований и задач, предъявляемых к соискателям.
Проанализировала емкость рынка по регионам и годам, обнаружила сезонные пики и спады, проанализировала
объем вакансий-стажировок и их распределение по регионам.| *pandas*, *re*, *numpy*, *nltk*, *pymystem3*, *plotly*, *matplotlib*, *seaborn*. |
| [Определение закономерностей, определяющих успешность игры](https://github.com/nyalekseevna/yandex_praktikum_projects/tree/master/game_platform_EDA) |
| В ходе исследовательского анализа данных выявила основные закономерности, определяющие успешность игры.
А именно, провела анализ жанров, популярных платформ, рейтингов целевой аудитории по регионам. 
Проверила статистические гипотезы распределения рейтингов игр по платформам и жанрам.| *pandas*, *numpy*, *scipy*, *matplotlib*, *seaborn*. |
| [Анализ результатов A/B тестирования](https://github.com/nyalekseevna/yandex_praktikum_projects/tree/master/AB_test) |
| Определила временной период для анализа, выделила события пользователя на сайте и их последовательность для построения воронки, построила продуктовую воронку с учетом последовательности событий. Проанализировала результаты А/В теста внедрения новой рекомендательной системы. Проверила гипотезы о статистической значимости изменения в конверсии.| *pandas*, *time*, *numpy*, *scipy*, *matplotlib*, *seaborn*, *ipywidgets*, *math*. |
